# QuickHyre AI - Full-Stack Talent Acquisition & HRMS Platform

QuickHyre AI is a scalable, full-stack web application designed to streamline end-to-end talent acquisition and workforce administration. The platform connects candidates with verified job opportunities while providing employers with an automated Applicant Tracking System (ATS) and HRMS suite to screen, onboard, and manage employees effectively.

## 🚀 Features

### For Candidates
* **Job Discovery:** Advanced searching and filtering across entry-level, mid-level, and senior tech/non-tech domains.
* **Seamless Application:** Profile creation and automated resume/application submission pipeline.

### For Employers (ATS & HRMS)
* **Job Management:** Post jobs, track applicant status, and screen resumes efficiently.
* **Workforce Onboarding:** Centralized dashboard to manage employee records and onboarding status.
